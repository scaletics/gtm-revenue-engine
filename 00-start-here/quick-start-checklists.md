## Purpose

Use these checklists to get a new GTM motion, sales campaign, client engagement, or partner initiative running quickly without overbuilding the operating system.

The goal is speed to execution with enough structure to avoid confusion.

---

## 2. New Sales Opportunity Checklist

Use before the first discovery call.

### Before the Call

- [ ] Confirm segment
- [ ] Confirm company fit
- [ ] Identify likely buyer
- [ ] Review segment pain points
- [ ] Prepare discovery questions
- [ ] Prepare two relevant business outcomes
- [ ] Prepare likely objections
- [ ] Define the desired next step

### During the Call

- [ ] Capture current state
- [ ] Capture desired future state
- [ ] Quantify business impact
- [ ] Identify decision process
- [ ] Identify stakeholders
- [ ] Confirm urgency
- [ ] Confirm budget posture
- [ ] Agree on next step

### After the Call

- [ ] Update CRM
- [ ] Score qualification
- [ ] Send relevant follow-up
- [ ] Schedule next meeting
- [ ] Identify missing information
- [ ] Determine whether proposal is appropriate

## 4. MSP Outbound Campaign Checklist

Use when launching an MSP-focused outbound campaign.

### Strategy

- [ ] Target MSP segment selected
- [ ] Service category selected
- [ ] Buyer persona selected
- [ ] Trigger event defined
- [ ] Offer angle selected
- [ ] Qualification criteria confirmed

### List Building

- [ ] Account list created
- [ ] Owner, CEO, GM, or sales leader identified
- [ ] Geography confirmed
- [ ] Service focus confirmed
- [ ] Customer base or employee range checked
- [ ] Exclusions removed

### Messaging

- [ ] MSP value proposition selected
- [ ] Email sequence customized
- [ ] Call opener prepared
- [ ] LinkedIn message prepared
- [ ] Objection responses reviewed
- [ ] One-pager prepared, if needed

### Execution

- [ ] Sequence launched
- [ ] Calls scheduled
- [ ] Replies categorized
- [ ] Objections logged
- [ ] Qualified leads scored
- [ ] Weekly review scheduled

## 5. Weekly Repo Maintenance Checklist

Use every week during active GTM execution.

- [ ] Add new objections heard in sales calls
- [ ] Update messaging if buyers repeat new language
- [ ] Capture competitor mentions
- [ ] Update qualification criteria if bad-fit leads appear
- [ ] Add new proposal lessons learned
- [ ] Archive stale drafts
- [ ] Update changelog for meaningful changes

## Operating Rule

If an asset does not help the team sell, qualify, position, propose, partner, research, or deliver client work, do not add it to the active repo.
